# Shopify Campaign Landing Page Assessment

## Overview

This repository contains my implementation of the Shopify Campaign Landing Page assessment.

The page has been developed using Shopify OS 2.0 best practices with reusable sections, modular assets, and a responsive layout.

---

## Features

- Responsive campaign landing page
- Banner section
- Product gallery section
- Quick View product modal
- Variant selection
- Quantity selector
- AJAX Add to Cart
- Theme Editor configurable settings
- Mobile-first responsive design
